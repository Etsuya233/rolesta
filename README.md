# Rolesta

## Development

开始开发前请阅读 [Rolesta 开发入口](AGENTS.md) 和 [开发工作流](docs/development/development-workflow.md)。需要 Git worktree 时，统一创建在仓库根目录的 `./.worktrees` 下。

## 项目介绍

Rolesta 是一个面向角色扮演聊天的 TypeScript monorepo，提供一对一聊天、兼容资产管理和上下文调试。平台支持导入和管理 SillyTavern 角色卡、世界书与预设，并围绕 Persona、模型连接、聊天偏好和会话组织创作流程。

产品面向个人部署，强调兼容内容可继续编辑、用户资产可控、生成上下文可解释，以及桌面和窄屏设备上的核心聊天体验。

## 当前产品边界

当前范围聚焦一对一角色扮演聊天、角色卡、世界书和预设的兼容导入与管理、模型连接配置，以及基础上下文组装和调试能力。发送、重新生成、编辑消息、切换预设和查看调试摘要等核心流程需要支持桌面与窄屏设备。

当前完成范围不包括完整复刻 SillyTavern、实时多人群聊、公开插件市场、强隔离插件沙箱、计费、组织权限、审计系统和原生移动 App。
